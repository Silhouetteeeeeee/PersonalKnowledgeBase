# 个人知识库 Agent
## 要求
1. 能把用户每日零碎的提问整合提炼成自己的知识，并且对问题进行分类存储，问题涉及的范围可能很广，可能是编程方面的疑问，也可能是生活中的困惑，你需要作为一个合格的倾听者，并且记住用户的生活习惯以及性格特征
2. 如果不了解用户提出的问题，你需要自己联网搜索进行学习（自我学习，这功能优先级可能略低）。并且对于问题提炼分类错误的，需要记录用户手动修改的记录，吸取这次错误的教训
3. 能够将用户上传的文档、笔记、图片记忆，并且学习，初始化自己的知识库
4. 有独立意识，不能用户说什么就是什么，不管结果正确性
5. 能够将知识库以思维导图的方式展示出来，方便用户查看阅读，后台最好存储结构化数据，方便可以引入第三方工具进行图形化

## 项目核心定位
本项目为私人专属智能提问管理Agent，属于轻量化智能工具类项目，无对外商业化属性，仅服务于单个用户的提问记录、知识沉淀与分类管理需求。核心定位是成
为用户的“私人提问管家+个人知识库构建工具”，通过自动化技术替代手动整理，将零散提问转化为结构化、可检索可视化的个人知识体系，支持长期维护与迭代更新

## 使用技术
Python + LangGraph
